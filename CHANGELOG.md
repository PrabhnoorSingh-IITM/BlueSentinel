# Changelog

All notable changes to the **BlueSentinel** project will be documented in this file.

## [V2.4] - 2026-03-03

### Fixed

- Corrected logo filename typo (`logon.svg` to `logo.svg`) and updated all references to use SVG.
- Fixed duplicate CSS stylesheet links in `logs.html`.
- Removed placeholder `alert()` calls from `archive.html`.
- Cleaned up redundant `console.log` statements in `dashboard.js`.
- Restored stability and surgically applied live data updates.

## [V2.0 - V2.3]

### Major Features Added

- **Major UI-UX Overhaul**:
  - Implemented "Bento Grid" modular layout for the dashboard.
  - Added Mac-style floating dock with magnification effects.
  - Implemented Glassmorphism design system (frosted glass, neon accents).
  - Integrated `Globe.gl` for 3D geospatial visualization of river incidents.
  - Added real-time Gemini AI health reports and remediation advice.
