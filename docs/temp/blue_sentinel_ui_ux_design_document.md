# BlueSentinel — UI / UX Design Document

## Design Philosophy
BlueSentinel’s interface reflects scientific monitoring seriousness blended with futuristic environmental intelligence.

Tone:
• Institutional  
• Analytical  
• Futuristic  
• Environmental

---

## Color Scheme

Primary Cyan: #00FFD4  
Primary Blue: #5465FF  
Light Blue: #D2DDFF  
Dark Background: #050208

Usage:
• Accents & highlights → Cyan  
• Buttons & gradients → Blue  
• Secondary text → Light Blue  
• Main backgrounds → Dark
