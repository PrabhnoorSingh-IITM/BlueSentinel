Technology Stack & Architecture

Hardware Layer

ESP32 Dev Module Primary microcontroller for sensor data collection
DS18B20 Temperature Sensor (Pin D4)
pH Sensor Module Water pH monitoring (Pin D32)
Turbidity Sensor Water clarity measurement (Pin D35)
WiFi Module Built-in WiFi for cloud connectivity

Cloud Infrastructure

Firebase Realtime Database Core data storage and real-time synchronization
Firebase Authentication Secure device and user authentication
Firebase Hosting Web deployment and CDN
Firebase Cloud Functions Serverless backend processing

Frontend Technologies

HTML5 / CSS3 / JavaScript Vanilla web development, no frameworks
Chart.js v4.4.0 Beautiful responsive live graphs and visualizations
Firebase SDK v9.22.0 Real-time data synchronization and authentication
Glass Morphism Design Modern frosted glass effects and animations

External APIs & Services

NewsAPI.org Marine conservation and ocean news aggregation
Twilio (Planned) SMS notifications for authorities and alerts
Weather APIs (Planned) Environmental data integration
